# Attendance Calculator 🎓

A modern, clean, and extremely fast web application to track university/school attendance. Find out exactly how many classes you can safely bunk, or how many you desperately need to attend to keep your percentage above your target goal!

## ✨ Features

- **Real-Time Calculations:** As you type, the formulas instantly compute whether you are in the safe zone or the danger zone.
- **Smart Tracking:** 
  - 🟢 Tells you the exact number of classes you can **safely miss** if you are above target.
  - 🔴 Tells you the exact number of classes you **must attend** consecutively if you are falling behind.
- **Dark Mode Support:** A beautiful, eye-friendly dark mode accessible via a single toggle button.
- **Minimalist UI:** Built with an Apple-inspired design aesthetic—focusing on plain colors, subtle shadows, and crisp typography. No clunky gradients or distracting backgrounds.
- **Animated Counters:** Smooth loading and transitional UI scaling based on pure mathematical React hooks. 
- **Lightweight Architecture:** Uses zero unnecessary visualization libraries, built strictly on React and vanilla CSS.

## 🚀 Built With

- **[React](https://reactjs.org/)** - For dynamic component states
- **[Vite](https://vitejs.dev/)** - Next-generation speedy frontend tooling
- **Pure CSS/JS** - Used for the visual framework and calculations without bloat

## 🛠️ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
You need Node.js and NPM installed on your machine.
- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repository
   ```sh
   git clone https://github.com/Blessd27/Attendance-Calculator.git
   ```
2. Navigate to the project directory
   ```sh
   cd attendance-calculator
   ```
3. Install NPM dependencies
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

## 📱 Responsiveness

This web app is 100% mobile responsive. The single card layout collapses gracefully to ensure the input form and dynamic status messages are perfectly legible on smartphones and tablets.
